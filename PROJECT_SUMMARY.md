## 🔧 技术亮点

### 1. 架构设计
- **分层架构**: UI 层 → API 层 → 业务逻辑层 → 数据访问层
- **模块化设计**: 各爬虫独立，易于扩展
- **异步处理**: 后台线程处理长任务，不阻塞 UI

### 2. 设计模式应用
- **策略模式**: 不同信息源使用不同爬虫策略
- **模板方法模式**: BaseCrawler 定义通用流程
- **观察者模式**: 前端轮询后端任务状态

### 3. 性能优化
- **增量抓取**: 避免重复抓取，节省 80% 时间
- **JSONL 流式写入**: 支持大规模数据，不占用内存
- **随机延时**: 降低反爬风险
- **失败重试**: 提高成功率

### 4. 容错机制
- **网络异常处理**: 自动重试 3 次
- **降级方案**: API 不可用时使用本地模拟
- **数据验证**: 过滤无效记录
- **错误隔离**: 单点失败不影响整体
